💎 Cartier-Inspired Website

A modern, responsive web application inspired by the timeless elegance of Cartier.
This project focuses on delivering a premium user experience through clean design, smooth interactions, and high-performance frontend architecture.

✨ Features
Luxury-inspired UI with a minimalist and elegant design
Fully responsive layout (mobile, tablet, desktop)
Optimized performance using Next.js features
Reusable and scalable component architecture
Smooth animations and transitions
Product showcase with detailed views

🛠️ Tech Stack
Framework: Next.js
Language: TypeScript
Styling: Tailwind CSS

🎯 Purpose

This project was built to improve frontend development skills by recreating a real-world luxury brand experience, with a focus on performance, responsiveness, and modern web practices.
